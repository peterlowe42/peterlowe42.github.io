Handy Enumerable Methods for The Ruby Beginner

Ruby has a huge range of handy methods contained in the Enumerable module. These methods can be called on any collection object including arrays, hashes, ranges and sets. As a ruby beginner by far the most common collection objects you will be working with will be arrays and hashes so this cheat sheet will focus on these. All the Enumerable methods are based on '.each' which simply iterates through the collection and passes each item to a code block, '.each' does not do anything with the results of this code block unless specificly told to. In the box below I have designed an array of numbers an array of words and a hash. Below that is a handy table showing you what several of the most common enumerable methods do. The method calls are shown on the left and the results are on the left.

Variable Definitions

  nums = [2,3,4,1,6,9,10,8,5,7]
  words = ["superman","is","boring","and","lame"]
  hash = {england:"London", fance:"Paris", germany:"Berlin", spain:"Madrid"}

Method Calls

	
  result = []
  nums.each {|i| result.push(i+1)}	   		
			
				
  nums.map {|i| i*2}

  words.map {|word| word.upcase}

  hash.map {|key,value| "#{value} - #{key.capitalize}"}
			    
				

  nums.select {|num| num.even?}

  words.select {|word| word.length > 4}

  hash.select {|key,value| value.length > 5
				

  nums.max

  nums.max(2)

  words.max {|a,b| a.length <=> b.length}
				

  nums.min

  nums.minmax
				

  nums.sort

  words.sort

  words.sort_by {|word| word.length}
