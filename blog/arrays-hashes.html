Arrays and Hashes

05/12/15

This post will attempt to define and compare arrays and hashes in programming languages. All the code and specific syntax I use will be for hashes and arrays in ruby but the basic concepts apply to many other languages.

What are Arrays?

Arrays are essentially containers for many items of data. They can contain numbers, words, symbols basically anything you might want a computer program to remember can be stored in an array. You can add and remove items as you see fit. In ruby arrays are represented by square brackets with each individual element inside separated by commas e.g [2,4,5,3]. Each element in the array is assigned an index number starting from 0 and can be called by typing array_name[index_num]. For example consider an array 'planets' containing the names of the planets in the solar system in order of distance form the sun. 

          
            planets = ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune', 'Pluto']

(I don't care what the haters say Pluto will always be a planet to me)

In this array 'Mercury' has position 0 and Earth has position 2. Starting to count from 0 may seem strange at first but you get used to it. If you wanted to print the 5th planet from the sun (Jupiter) you would type 

         
           puts planets[4]    # => Jupiter

Note '4' is the index number for the fifth element in the array.

What are Hashes?

Arrays are great but you may have noticed that you can only call a specific element from an array if you know what order they appear in. What if you want to store a large number of items in no specific order and still be able to find the one you want. This is where hashes come in. Hashes are a lot like arrays but instead of index numbers each 'value' is assigned a 'key' and as long as you know the name of that key you can call the value you want without having to know where it appears in the hash.

For example if you wanted to store a list of super heroes powers a hash would be perfect. You could use the name of the super hero as the 'key' and their power as the 'value'. You'd write it like this 

        
          super_powers = {
             'Spider-man' => 'Does whatever a spider can'
             'Hulk' => 'Turns into giant green rage monster'
             'Nightcrawler' => 'Teleportation'
             'Batman' => 'Money'
             'Prof. X' => 'Telpathy'
        }

Now if you want to print the Hulk's power you could just type: 

          
            puts super_powers[Hulk]    # => Turns into giant green rage monster

You could do the same with any of the super-heroes in the hash without having to know the order in which they appear. Another great thing you can do with hashes is assign a default value that will be returned if you attempt to look up a key that is not in the hash.

I hope this has helped to explain the concepts of arrays and hashes and why you might want to use them. Happy coding.....
